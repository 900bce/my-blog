React 新手上路(一)

Posted on 2019-10-02 Edited on 2019-10-03

React 思維：

有別於以往JS邏輯將運算和輸出寫在一起，React 的操作方式是將畫面呈現和資料輸出分開，用Components的概念將介面定義好後，再把資料輸進去，每當“狀態”改變就重新render。

React 語法：

當我們想要在畫面上放一個

<div>Hello World!</div>

React 中我們這樣寫

// Hello.js

import React from 'react';

function Hello() {
  return <div>Hello World!</div>
}

export default Hello;

這樣就是一個React compoent了

注意! Component 的名稱開頭必須為大寫。

再來我們試著將 Hello 這個 component 放到畫面上

<!-- index.html -->
<body>
  <div id="root"></div>
</body>

// index.js

import React from 'react';
import ReactDOM from 'react-dom';
import Hello from './Hello';

ReactDOM.render(<Hello />, document.getElementById('root'));

這樣就完成了！

Props:

當component的上層需要傳遞資料給component，使用props。

// Hello.js

import React from 'react';

const Hello = (props) => {
  return <div>{props.hello}</div>
}

export default Hello;

// index.js

import React from 'react';
import ReactDOM from 'react-dom';
import Hello from './Hello';

ReactDOM.render(<Hello hello={'Hello World!'}/>, document.getElementById('root'));

STATE:

我們已經知道怎麼丟props給components,
但component要如何和其他component互動呢？

利用STATE

STATE >> props

當parent改變狀態(state)，再把state轉為props丟給下層，因為component只能接收props再return結果。

使用class component:

import React, { Component } from 'react';

class Hello extends Component {
  constructor() {
    super();
    this.state = {
      text: 'Hello World!'
    }
  }

  changeText = () => {
    this.setState({
      text: 'Hello React!'
    })
  }

  render() {
    return (
      <div conClick={this.changeText}>
        <HelloText text={this.state.text} />
      </div>
    )
  }
}
