React deploy 到 Github page

Posted on 2019-10-21

如何將 React 部署到 Github page 上

create react app 有提供工具可以直接將 React 專案部署到各個 server 上（AWS, firebase, GH-page,…等等）。 既然Github是每個人都有用而且也是免費的，我想應該是最方便的靜態網站平台。所以今天來練習如何部署React專案到Github上。

參考(https://create-react-app.dev/docs/deployment/)

第一步：

這一步非常重要
將以下寫進專案的 package.json 中。

"homepage": "https://myusername.github.io/my-app",

第二部：安裝 gh-pages 以及 加入 deploy scripts 到package.json裡面

npm install --save gh-pages

接著在package.json裡面加入：

"scripts": {
+   "predeploy": "npm run build",
+   "deploy": "gh-pages -d build",
    "start": "react-scripts start",
    "build": "react-scripts build",

其中 predeploy 會在 deploy 時自動被執行。

第三步：

npm run deploy

第四部：

在github repo的設定裡，找到github page部分的 source選項，確定有選到 gh-pages branch， 就大功告成了。

React 新手上路(一)

Posted on 2019-10-02 Edited on 2019-10-03

React 思維：

有別於以往JS邏輯將運算和輸出寫在一起，React 的操作方式是將畫面呈現和資料輸出分開，用Components的概念將介面定義好後，再把資料輸進去，每當“狀態”改變就重新render。

React 語法：

當我們想要在畫面上放一個

<div>Hello World!</div>

React 中我們這樣寫

// Hello.js

import React from 'react';

function Hello() {
  return <div>Hello World!</div>
}

export default Hello;

這樣就是一個React compoent了

注意! Component 的名稱開頭必須為大寫。

再來我們試著將 Hello 這個 component 放到畫面上

<!-- index.html -->
<body>
  <div id="root"></div>
</body>

// index.js

import React from 'react';
import ReactDOM from 'react-dom';
import Hello from './Hello';

ReactDOM.render(<Hello />, document.getElementById('root'));

這樣就完成了！

Props:

當component的上層需要傳遞資料給component，使用props。

// Hello.js

import React from 'react';

const Hello = (props) => {
  return <div>{props.hello}</div>
}

export default Hello;

// index.js

import React from 'react';
import ReactDOM from 'react-dom';
import Hello from './Hello';

ReactDOM.render(<Hello hello={'Hello World!'}/>, document.getElementById('root'));

STATE:

我們已經知道怎麼丟props給components,
但component要如何和其他component互動呢？

利用STATE

STATE >> props

當parent改變狀態(state)，再把state轉為props丟給下層，因為component只能接收props再return結果。

使用class component:

import React, { Component } from 'react';

class Hello extends Component {
  constructor() {
    super();
    this.state = {
      text: 'Hello World!'
    }
  }

  changeText = () => {
    this.setState({
      text: 'Hello React!'
    })
  }

  render() {
    return (
      <div conClick={this.changeText}>
        <HelloText text={this.state.text} />
      </div>
    )
  }
}
